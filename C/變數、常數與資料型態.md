### 輸入的格式特定字元
```
int = %d
double,float = %f
```
```
/* output40.c */
#include <stdio.h>
#include <stdlib.h>
int main()
{
    printf("我目前有 %d 元\n", 100);
    printf("我的C語言成績是 %f 分\n", 89.98);
    system("PAUSE");
    return 0;
}
```
